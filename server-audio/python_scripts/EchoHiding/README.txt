Данное приложение позволяет скрыть в аудиофайле небольшое сообщение и затем извлечь его, причем сокрытие производится путем наложения эха, неразличимого для человеческого слуха. 

На вход приложение принимает файлы формата .wav, .mp3, .mp4, однако для работы с двумя последними требуется дополнительно загрузить программу FFmpeg(http://ffmpeg.org/) и прописать путь до исполняемого файла в переменных среды. 

В папке test содержится пример результата использования приложения.

Надо, однако, принимать во внимание, что специфичность метода приводит к некоторой неустойчивости, и алгоритм может совершенно не подойти для аудиофайлов, изначально сильно зашумленных.
